# ConvoHi - Full-Stack Real-Time MERN Chat Application

ConvoHi is a production-ready, feature-rich real-time chat application designed for modern web communications. Built on the MERN stack, it leverages Socket.IO to enable instant message delivery, real-time typing indicators, user presence detection, message reactions, and threaded replies.

## 🚀 Key Features
*   **Secure Authentication**: JWT-based session state, password hashing using `bcryptjs`, and protected React Router guards.
*   **One-to-One Conversations**: Instant messaging with real-time delivery and visual read receipts (seen ticks).
*   **Group Chat Rooms**: Create persistent group spaces, manage members, assign admins, and check group metadata.
*   **Media Sharing**: Share images, PDFs, and text documents securely processed and hosted via Cloudinary integrations.
*   **Interactive Controls**: Threaded replies, soft-deletions, editing sent messages, and instant message reactions (👍❤️😂🔥).
*   **Presence Indicators**: Live online/offline tracking, last-seen timestamps, and real-time typing state.
*   **Responsive Dark/Light UI**: Tailored using custom Tailwind CSS variables for maximum performance and visual appeal.
